# Use cases — Elite plan (₹1,599/mo)

Elite is **Growth plus paid-social depth**: **Social ads** (Meta credentials, drafts, visuals), **ad creative angles** inside **Site audit**, and **monthly email with PDF** attachment when configured.

---

## Primary journeys

1. **Paid + organic alignment**  
   Run **Site audit** → copy **Ad creative angles (Meta / Google RSA-style)** into Ads Manager alongside organic recommendations.

2. **Social automation**  
   Store **Meta credentials JSON** on **Social ads** (see on-page tooltip for token overview) → review **Recent drafts** → copy packs / creative notes for publishing.

3. **Full Growth workflows**  
   Blogs, backlinks, GA4, GSC, competitors (up to **5**), calendar, shareable reports, monthly email **with PDF** for Elite.

4. **Local business**  
   Enable **local SEO pack** on **Business profile** and use local vertical/goal so audits add GBP-oriented drafts (posts, Q&A, NAP, review snippets).

---

## Elite-only surfaces

| Surface | Purpose |
|---------|---------|
| `/dashboard/social` | Meta JSON + ad draft gallery |
| Site audit — Ad creative angles | Headline/body variants for Meta & Google RSAs |
| Monthly email | Same as Growth plus **PDF** brief when cron + Resend are set |

---

## Credentials

- **Meta:** Page access token + Instagram Business Account ID in JSON (see UI tooltip).  
- **Google:** Same OAuth app as GSC for GA4; optional BYOK Gemini/PageSpeed on **Business profile**.
