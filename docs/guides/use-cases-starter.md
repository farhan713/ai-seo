# Use cases — Starter plan (₹499/mo)

Starter clients get **full SEO core**: audits, crawl, Search Console, keywords, calendar (small cap), on-page checklist, competitors (1), and dashboard growth score.

---

## Primary journeys

1. **Onboard**  
   Register → complete **Business profile** (industry vertical, marketing goal, URL, keywords) → presets flow into AI audits and content.

2. **Measure search**  
   Connect **Search performance** (Google OAuth) → sync queries → review **Keywords** (auto-seeded from profile + GSC).

3. **Improve the site**  
   Run **Site audit** → apply CTA/FAQ/lead-magnet blocks → use **On-page checklist** per URL → optional **Site crawl**.

4. **Plan content**  
   **Content calendar** (up to 2 items on Starter) → seed ideas from audit pillars where available.

5. **Social captions from audit**  
   Generate **social post pack** from an audit run → **LinkedIn-only** captions on Starter.

6. **Share progress**  
   **Share report** (read-only link) to send a snapshot to a stakeholder without account access.

---

## Not included on Starter

- AI **Blogs** and **Backlinks** checklist (Growth+)  
- **Web analytics (GA4)** (Growth+)  
- **Instagram/Facebook** social pack variants on blogs (Growth+)  
- **Social ads** automation and **ad creative angles** in audit (Elite)  
- Monthly executive **email** opt-in (Growth+)

---

## Optional client keys

- Clients may add **Gemini** or **PageSpeed** keys under **Business profile** if you want per-tenant quota or no shared server key.  
- **Local SEO pack** toggle: when on and the profile fits local intent, audits can include GBP-style drafts.
